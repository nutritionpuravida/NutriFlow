# NutriFlow

NutriFlow is a private clinical nutrition web app for dietitians.

It includes client profiles, exchange-based nutrition planning, 1-day and 7-day meal plans, blood analysis support, progress tracking, recipes, and printable client reports.

## Deployment

This repository is intended to be deployed with GitHub Pages from the root `index.html` file.

Target URL:

https://nutritionpuravida.github.io/nutriflow

## Privacy

This is a clinical nutrition planning tool. Do not publish real client data, credentials, API secrets, or protected health information in this repository.
